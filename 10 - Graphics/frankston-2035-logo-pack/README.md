# Frankston 2035 - Art That Lives Logo and SVG Pack

Independent civic-art identity pack for the Frankston 2035 project.

This pack gives the proposal its own usable visual identity: a bay wave, pier line, sunrise/sand colour, foreshore strip, and clear Frankston 2035 typography. It is designed to support posters, web pages, school documents, social previews, and council-facing drafts without pretending to be official Frankston City Council branding.

## Start Here

Use the independent Frankston 2035 assets for public-facing project material.

Do not use the Frankston City logo files in `reference-only/` unless council has explicitly approved that usage. Those files are included only as visual reference DNA, not as active campaign assets.

## What Is Included

- `svg/` - editable vector artwork for lockups, wordmarks, stacked marks, and standalone marks.
- `png/light/` - PNG exports for use on light or white backgrounds.
- `png/dark/` - PNG exports for use on dark backgrounds.
- `png/transparent/` - transparent-background PNGs for flexible layout use.
- `social/` - square social icons and Open Graph card artwork.
- `icons/` - favicon and app-icon exports.
- `web/site.webmanifest` - web manifest for installable/static-site use.
- `palette/` - colour tokens as CSS and JSON.
- `reference-only/` - Frankston City logo references supplied for comparison only.
- `frankston-2035-logo-pack-preview.jpg` - quick visual overview of the pack.

## Which File Should I Use?

### Website Header

Use:

- `svg/frankston-2035-lockup-transparent.svg`
- or `png/transparent/frankston-2035-lockup-transparent-2048.png`

Best when the logo sits on a light, textured, or image-based background.

### Dark Poster Footer

Use:

- `png/dark/frankston-2035-lockup-dark-2048.png`
- or `svg/frankston-2035-lockup-dark.svg`

Best for poster footers, dark title cards, and presentation slides with deep navy or ink backgrounds.

### Light Document Cover

Use:

- `png/light/frankston-2035-stacked-light-1600.png`
- or `svg/frankston-2035-stacked-light.svg`

Best for a title page, one-page summary, or council-facing PDF cover.

### Small Icon or Avatar

Use:

- `social/frankston-2035-social-icon-dark.png`
- `social/frankston-2035-social-icon-light.png`
- `icons/favicon.svg`
- `icons/favicon.ico`

Best for browser tabs, social avatars, folder icons, and small UI marks.

### Inside Documents

Use:

- `svg/frankston-2035-wordmark-transparent.svg`
- or `png/transparent/frankston-2035-wordmark-transparent-1600.png`

Best when the mark has already appeared once and you only need a quiet running identity.

## SVG, PNG, Or Favicon?

Use SVG when the artwork needs to stay sharp at any size, such as websites, print layout, signage drafts, or editable design files.

Use PNG when placing artwork into tools that handle images more reliably than vectors, such as Word, Google Docs, PowerPoint, email, or social posts.

Use favicon files only for browser tabs, app icons, and web manifests. They are not poster or document assets.

The palette should feel coastal, civic, public, and warm. Avoid making the whole system one-note blue. Use sand gold and sunset coral sparingly for emphasis.

## Brand Safety

Frankston 2035 is an independent proposal identity. It should look credible and civic, but not official.

Do:

- Keep Frankston as the visual hero.
- Use the Frankston 2035 lockup as the project identity.
- Keep Blue Snake Studio as the maker stamp or delivery engine.
- Leave enough clear space around the mark.
- Use high-contrast versions on busy backgrounds.

Do not:

- Use the official Frankston City logo from `reference-only/` on public assets without approval.
- Make the project look like an official council service.
- Put the mark over unreadable imagery.
- Stretch, squash, recolour, or add effects to the mark.
- Let Blue Snake Studio overpower the Frankston 2035 identity.

## Quick Production Notes

- For web, prefer SVG unless file-size or CMS limitations require PNG.
- For A4 posters, use PNG at the largest supplied size or place the SVG directly in the design file.
- For Word or Google Docs, PNG is usually safer.
- For social previews, use the supplied 1200 x 630 Open Graph cards.
- For colour consistency, import `palette/frankston-2035-palette.css` or use the JSON tokens in `palette/frankston-2035-palette.json`.

## Plain-English Rule

If the asset is representing the Frankston 2035 proposal, use this pack.

If the asset is representing Frankston City Council, stop and get explicit approval first.
