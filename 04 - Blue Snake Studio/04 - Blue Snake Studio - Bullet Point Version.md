**The question:** Can this actually be built?

**The proof — existing, working projects:**
- Digital DNA Laboratory → interactive browser-based learning, 5 play modes, already solved
- Nara's Colour Quest → kid-safe, privacy-first design, by default not by policy
- Hepta Lock → tested, verified logic engine — rigour, not just visuals
- Moss60 → original base-60 maths system — real substance behind "art that teaches maths"
- Black Wing Crew / Susono exchange → the sister-city bridge already exists

**What it maps to in the Frankston proposal:**
- Interactive learning layer → solved
- Child safety/privacy → solved
- Tested logic, not just decoration → solved
- Mathematical content engine → solved
- Susono cultural bridge → already real

**What this is NOT:**
- Not a claim any of this was built for Council
- Not a claim it deploys unmodified
- Genuine adaptation work still required

**Why it lowers risk:**
- Funding adaptation of proven work ≠ funding invention of something new
- Hard technical problems already solved
- What remains is application to Frankston specifically

**The handoff:**
- Section 01's pilot pathway = the next proportionate step
- Proves adaptation, not invention, is the actual ask
