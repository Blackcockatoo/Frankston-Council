# 04 — Blue Snake Studio
## Visual Layout Suggestions

---

**Opening page**
- Memetic primary line as section header pull-quote ("The hard part is already built...") — this is the one page in the whole package where B$S is allowed visual confidence, since its entire job is proving capability.
- Small, secondary framing line beneath: "Evidence, not introduction." Keeps the tone in line with Constitution Section 4 (earned trust over big claims).

**Portfolio grid page**
- Five-item grid (Digital DNA Lab / Nara's Colour Quest / Hepta Lock / Moss60 / Black Wing Crew-Susono), each with: one screenshot or simple representative graphic, a one-line caption, and — critically — a second line in the data/mono face stating what it *proves for Frankston* (not just what it is). This second line is what keeps this page from reading as a generic portfolio and keeps it tied back to Section 01.
- Use `bay-pale` panel backgrounds per item, consistent hairline borders, per Asset Standards.

**"What this proves" table page**
- Render the Main Document Draft's mapping table (existing work → Frankston capability) as an actual two-column visual table, `bay-pale` header row, generous padding — this table is the single most important asset in this section and should get its own page, not be squeezed under prose.

**"What this is not" page**
- Treat as a quiet, deliberately undecorated page — plain text, generous white space, no icons. The honesty of this section should visually read as calm and confident, not defensive. Per Constitution Section 5, honesty is a brand feature, so it shouldn't look like a disclaimer buried in fine print.

**Closing bridge page**
- Mirrors Section 01's closing bridge page in reverse: where 01 ends pointing forward to 04, 04 ends pointing back to 01's pilot pathway. Use the same coral call-to-action box treatment for visual consistency across the two documents when bound together as the Intro Pack.

**Don'ts specific to this document:**
- Don't use B$S's sacred-geometry visual motifs here even though this is the one B$S-led section — Branding Rule (Section 8) still applies; civic-first framing holds even in the credibility section.
- Don't let the portfolio grid become five disconnected case studies — every caption must close the loop back to "what this proves for Frankston," or it reads as a generic agency portfolio.
