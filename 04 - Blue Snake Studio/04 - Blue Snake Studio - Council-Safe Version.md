# 04 — Blue Snake Studio
## Council-Safe Version

Suitable for direct inclusion in council briefing material without further political review.

---

**Purpose of this section**

To provide Council with an evidence basis for assessing whether Blue Snake Studio has the technical capability to deliver the pilot described in Section 01, independent of any claims made elsewhere in this proposal.

**Existing, independently-built body of work**

Blue Snake Studio has, over approximately two years, designed, built, and tested the following projects, none of which were commissioned by or built for Frankston City Council:

- An interactive web-based learning platform (multiple play modes, browser-based, no installation or login required, no user data collection)
- A child-directed learning website built with privacy-by-design as a foundational requirement
- A logic puzzle engine with verified, tested game logic
- An original mathematical framework underpinning the above projects
- An existing creative project connected to the genuine, pre-existing Frankston–Susono sister-city relationship

**What this evidence does and does not establish**

This evidence establishes that the studio has relevant technical capability and a demonstrated design philosophy consistent with the proposal's stated commitments (interactivity, child privacy, mathematical rigour, cultural connection). It does **not** establish that any existing project can be deployed to Council unmodified, nor does it constitute a guarantee of delivery timeline or cost for a Frankston-specific pilot — those details belong in Section 08 (Pilot Projects) and any resulting procurement documentation.

**Relevance to risk assessment**

Council officers assessing this proposal may find it relevant that the core technical risks associated with the proposed pilot (interactive content delivery, child data handling, content rigour) appear to already have working precedents in the studio's existing portfolio, which may reduce — though does not eliminate — delivery risk relative to a proposal with no comparable prior work.
