# 08 - Pilot Projects
## 1-Page Summary

Frankston 2035 - Art That Lives should begin with one small pilot, not a full program.

The five pilot options are:

| Pilot | Plain-English version | Why it matters |
|---|---|---|
| Living Mural / QR Public Art Layer | One mural or public location gets a no-login learning layer. | Tests the core idea visibly: public art that teaches. |
| Library Creative Pack | A printable/digital pack families and teachers can use immediately. | Moves the idea into a safe, low-cost community setting. |
| Frankston x Susono Classroom Exchange Pack | A review-ready classroom resource for sister-city learning. | Makes the relationship usable, but does not claim official participation. |
| Healthy Technology Mini-App | A short, child-safe creative web app with offline prompts. | Proves technology can return attention instead of capturing it. |
| Respect Through Participation Pathway | A protocol-first pathway for future First Nations learning work. | Protects the project from tokenism and overclaiming. |

The strongest first candidate is the Living Mural / QR Public Art Layer. It is concrete, visible, and easy to keep small. A single existing artwork or public location can be paired with one QR code, one no-login web page, one learning activity, and one feedback loop. The pilot can run for 30/60/90 days and be stopped, revised, or expanded based on evidence.

The main safeguard is restraint. No pilot should imply council adoption, First Nations permission, Susono participation, school endorsement, or long-term funding unless those things are real.

The minimum viable first ask is:

> Approve exploration of one small scoped pilot.

The next decision is not "Should Frankston fund the whole vision?" The next decision is: which one pilot is safe enough, useful enough, and clear enough to scope first?
