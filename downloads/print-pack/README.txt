Print notes:
- Use A4 portrait.
- Use "scale to fit" if printing from a browser.
- Turn on background graphics.
- Print one poster per page.
- SVG files are 2480 x 3508 pixels, matching A4 ratio.

Identity/SVG pack:
- Use the Frankston 2035 independent mark for project materials.
- Do not use the official Frankston City logo from reference-only material unless council has approved that usage.
- See 10 - Graphics/frankston-2035-logo-pack/README.md for detailed usage guidance.
