# 08 - Pilot Projects
## Memetic / High-Impact Version

## Core line

Do not launch the vision. Test one living piece of it.

## Short lines

- One wall. One code. One lesson. One term.
- A pilot is a promise with brakes.
- Public art should teach before it scales.
- If it cannot be tested in 90 days, it is not the first move.
- Respect is not content. Respect is process.
- Technology should point children back to the wall, the walk, the page, and the person beside them.
- Sister-city exchange means reply, not decoration.
- The first win is not reach. The first win is trust.

## Poster-ready line

Start small enough to be honest. Build carefully enough to last.
