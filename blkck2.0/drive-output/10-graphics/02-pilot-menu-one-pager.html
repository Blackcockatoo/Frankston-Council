Frankston 2035 - Pilot Menu

Choose one low-risk first move.

Do not bundle all five. Pick one pilot, test it honestly, then decide whether to continue, revise, or stop.
Tiny: planning, copy, print material, lightweight prototype. Small: simple production, web layer, printing, or facilitation. Medium: multiple partners, permissions, approvals, or careful review.
Recommended first pick

Living Mural Lesson Wall

Purpose
One existing artwork becomes a low-prep learning surface.
Risk level
Medium: artist/site permission and public clarity.
Council ask
Nominate one candidate wall and advise the permission pathway.
Cost band
Small.
30 / 60 / 90
30: site choice. 60: QR lesson and worksheet. 90: limited test and go/stop/revise note.
Permissions
Artwork/copyright, site/signage, school participation, protocol gate if cultural content appears.
Visible civic test

Frankston Creativity Trail

Purpose
A short walk turns public space into creative prompts.
Risk level
Medium: wayfinding, accessibility, sign clutter.
Council ask
Choose three walkable stops and advise temporary signage rules.
Cost band
Small.
30 / 60 / 90
30: paper prompts. 60: map and QR pages. 90: small public or library walk.
Permissions
Temporary signage, image/artwork permissions, accessibility review, moderation rules.
School/library test

Scan Less / Notice More Walk

Purpose
A healthy-technology walk where scanning is brief and attention returns to place.
Risk level
Low: supervision and weather are the main constraints.
Council ask
Support one 20-minute test with a class, library group, or family walk.
Cost band
Tiny.
30 / 60 / 90
30: prompt cards. 60: landing page and worksheet. 90: trial walk and feedback.
Permissions
Site/event use if needed, school/guardian process, privacy rule: no personal data.
Later / review first

Sister City Song + Poster Exchange

Purpose
One replyable artifact between Frankston and Susono-facing participants.
Risk level
High: approval, translation, tokenism, timing.
Council ask
Identify the correct sister-city review contact before any public claim.
Cost band
Medium.
30 / 60 / 90
30: invitation brief. 60: draft only if review path opens. 90: exchange only if approved.
Permissions
Sister-city review, school/family publishing consent, translation check, pilot proposal label.
Good second move

Community-Made Resource Library

Purpose
A shared shelf of printable prompts, worksheets, and maker resources.
Risk level
Medium: copyright, moderation, privacy, quality control.
Council ask
Host one library or school table and review contribution rules.
Cost band
Small.
30 / 60 / 90
30: template set. 60: static page and moderation checklist. 90: table/workshop test.
Permissions
Contributor licence, child-safe publishing, privacy review, protocol gate for place-based content.

First pick: Living Mural Lesson Wall. It is visible, bounded, useful, and easiest for Council to understand.

Proof gate: no login, no child data, printable fallback, accessible mobile page, clear exit prompt.

Protocol label: protocol pathway only - no permission implied.
