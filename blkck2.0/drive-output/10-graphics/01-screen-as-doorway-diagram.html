Frankston 2035 - Healthy Technology

Screen as Doorway, Not Destination

Technology should give attention back to Frankston, people, and making.

Draft production diagram
Use for council-safe checklist and pilot review.
01
QR / Link

Opens the door without app install or default login.

02
Short Prompt

One clear task. Plain English. No infinite scroll.

03
Offline Action

Walk, notice, draw, discuss, print, make, or teach.

04
Optional Reflection

Adult-led or group-safe. Anonymous aggregate metrics only if needed.

05
Finish

The experience ends. The user returns attention to place.

Proof Gates

  • Load speed
  • No-login test
  • No-child-data test
  • Printable fallback
  • Mobile readability
  • Accessibility pass
  • Visible privacy note
  • Clear exit prompt
  • Adult-led group mode

Decision Rule

If it steals attention, cut it. If it needs child data, redesign it. If it encourages compulsive use, reject it.
