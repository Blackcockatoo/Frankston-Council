Student artwork pathway

From Classroom Work to Public Display

A safe pathway for student creativity. This is not a publishing permission form.

  1. Create in class using a local place, artwork, or community prompt.
  2. Teacher reviews for safety, respect, accessibility, and school policy.
  3. Family consent is gathered before any public display or online use.
  4. Display uses first name only, initials, class group, or anonymous credit where appropriate.
  5. Remove, revise, or pause if a student, family, school, or community reviewer asks.

Do not collect

Student addresses, personal contact details, private accounts, location histories, or unnecessary profile data.
