# Codex Audit Report - 2026-06-26
## Frankston 2035 - Art That Lives

## Executive summary

The project has a strong thesis, a useful constitution, and a credible council-facing premise: Frankston is the hero, Blue Snake Studio is the delivery engine, and the first ask should be one small scoped pilot. The weakness is not imagination. The weakness is operating discipline.

Before this pass, the Drive folder looked more complete than it was. Many folders existed with no content. Section 08, the pilot engine, was empty. The roadmap was philosophically sound but too light to guide action. The Contents index tracked sections but not ownership, dependencies, or next action. The seed kit existed, but could outrun review if treated as public-ready.

The biggest risk is overclaiming cultural and civic readiness before permissions, council pathways, and partner roles exist.

## Major risks found

1. **Cultural protocol risk:** First Nations content must not become content production. It is a permission-led process pathway only.
2. **Sister-city overclaiming risk:** Susono material must remain proposal language until reviewed by relationship custodians.
3. **Ask inflation risk:** The project can easily sound like a large program ask. It must stay at "approve exploration of one small scoped pilot."
4. **Proof gap:** Blue Snake Studio has strong proof claims, but the Drive package needs links/screenshots/evidence before external circulation.
5. **Seed kit risk:** The static site/poster pack is useful, but not council-reviewed and not protocol-cleared.
