# 00 - README
## Frankston 2035 - Art That Lives
### Start here

This folder is the control room for the proposal. Read this file first, then read `Project Vision.md`, `Foundation (Chunk 00 - Constitution)/Chunk 00 - Project Foundation.md`, `Contents.md`, and `Roadmap.md`.

If a later document contradicts the constitution, the constitution wins. If the constitution is wrong, fix it there first and then check every downstream file.

---

## What this project is

Frankston 2035 - Art That Lives is a civic creativity framework where Frankston's public art and creative culture become usable public infrastructure: something residents can learn from, contribute to, share, and build on.

The current ask is not a full rollout. The current ask is to prepare one safe, small, scoped pilot that council can understand, test, and evaluate without committing to the whole ten-year vision.

Blue Snake Studio is the prototype lab and delivery partner. Frankston is the hero.

---

## What this project is not

- It is not an announcement of council approval.
- It is not a claim of First Nations partnership or permission.
- It is not a claim of official Susono participation beyond Frankston's existing sister-city relationship.
- It is not a request to fund the whole vision now.
- It is not a tech platform looking for a civic excuse.
- It is not a Blue Snake Studio brand campaign with Frankston attached.

Any document that implies one of those things is a risk and needs revision.

## How to read the folder

1. Start with this README for orientation.
2. Read `Project Vision.md` for the thesis.
3. Read `Foundation (Chunk 00 - Constitution)/Chunk 00 - Project Foundation.md` for voice, naming, protocol, design, and claim rules.
4. Read `Contents.md` to see what exists, what is missing, and what is blocked.
5. Read `Roadmap.md` to understand the build sequence and decision points.
6. Read `01 - Executive Summary` and `04 - Blue Snake Studio` together. One explains the civic opportunity; the other proves capability.
7. Read `08 - Pilot Projects` before drafting speeches, decks, posters, funding copy, or public material.
