# Foundation — Bullet-Point Quick Reference

**Project:** Frankston 2035 — Art That Lives (Blue Snake Studio)

**One sentence:** Public art that teaches, travels, and turns viewers into creators — built on what Frankston has already invested in, not instead of it.

**Tone:** visionary / practical / civic / beautiful / community-first — never corporate, never critical of Council.

**Hard rules:**
- First Nations content = invitation language only, never partnership claims.
- Susono content = proposal language only, never "existing program" claims.
- No fake stats. No fake studies. No fake partnerships.
- No engagement-maximising metrics for kids' tools.
- B$S logo always smallest, always last, always below Council/Traditional Owner/Susono marks.

**Every document needs:**
- The single message → why it matters to Frankston → what already exists → what's proposed → how risk is contained → next small step.

**Palette:** bay-deep `#1B3A4B` · bay-mid `#3E7186` · bay-pale `#BFD7DD` · sand `#E8E1D3` · dusk-coral `#D98E73` (used once, maybe twice per page) · paper `#FAFAF7`.

**Type:** serif display for titles only · clean sans for body · mono/condensed for numbers and data.

**File naming:** `[Section] - [Name] - [Title] - [Depth].ext` — no version numbers in filenames.

**Source of truth hierarchy:** Chunk 00 (this) → Project Vision → Roadmap → Contents index → everything else.

**Golden rule:** when unsure, under-claim. It's easier to expand an honest proposal than retract an overclaimed one.
